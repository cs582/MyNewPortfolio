> Research Engineering Projects

Low-Resource Contrastive Language Image Pre-learning

Contrastive Language Image Pre-training (CLIP) is an algorithm developed by OpenAI that connects text with images. It makes use of Natural Language Processing (NLP) and Computer Vision to project them onto the same space. I worked on my own implementation of this algorithm, wrote the pseudocode for general implementation, implemented it from-scratch (PyTorch), built a new dataset of 24 million text-image pairs for this task, and trained on low-computing resources.

Recreation of Generative Adversarial Networks and Diffusion Models

Generative Adversarial Networks (GANs) and Diffusion Models are neural network models that are used to generate images and other media. These algorithms were very influential to build current state-of-the-art image generation algorithms such as DALLE-2, Stable Diffusion, and Midjourney. I implemented the original papers “Generative Adversarial Networks” and “Deep Unsupervised Learning using Non-equilibrium Thermodynamics” and trained them on MNIST.

Reinforcement Learning and Algorithms for High Frequency Trading

Reinforcement Learning uses Q-Learning to approximate the actions taken by an agent to the ideal actions it could take under a given environment, in this case, a trading environment. I evaluate Convolutional Neural Networks (CNNs) vs Vision Transformers (ViTs). I found that not only do ViT's outperform CNNs but they also take more steady strategies throughout.

> Software Engineering Projects

Cloud-Native Music Recommendation System

Built an API to make music recommendations using Playlists from the user. I built the front-end on Flask and the back-end on Lambda. The recommender consists of two microservices, the ml-model trainer and the recommendation system, both were running on the cloud. I used Kubernetes for orchestration and ArgoCD for continuous development.

Cloud-Native Web Monitoring Dashboard

Developed and deployed a real-time Web Monitoring dashboard. This service used Redis to store temporary information and Flask for the front-end. I developed the web-dashboard using D3.js to show and monitor Memory Use, CPU Use, and Network Use. In this case, I used a function as a service to make the calculations as well as ArgoDC and Kubernetes for continuous deployment and orchestration.
